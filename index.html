This is a project page. Your abstract will appear here. As Shreya as already decreed, it cannot be over 100 words. She’ll also edit your grammar and spelling, but she can’t make this section as fun as you can. Please put some effort into this bit. About the rest of the page- good news! No overall word limit or image-limit. We can put up as many pictures as your project needs. Just follow the prescribed image resolutions and naming format strictly. We want all our project pages to look really good, so go through the instructions carefully and comply. Your DDS headshot will appear on the left.  Photoshoot team will supply that, you can chill. Cover picture must be 1200 x 800 px.  Do not put anything important there (without repeating it elsewhere) as it might be cropped in different devices. Also provide a thumbnail of 300x300px. This page has not been optimised for mobile or tab viewing yet. We’ll fix that as we go. Also, you can add up to 5 separate academic projects this time. Not restricted to P1, 2 and 3. Images used here are for representation only. Our project images will be way better. No?

This text corresponds to images on the left. Images are in a sliding gallery. Left/right buttons will be added. One image set and corresponding text together are referred to as a section. This is section 1. You can have as many such sections as you like, just limit each to 100 words. The image set should be named Section_1_A, Section_1_B, etc. If you need more such sections, name the Section_2_A, Section_3_A and so on. Images need to be 600 x 400, strictly. Just make sure each section has some text to go with it, unless it’s full width. Refer to last section in this page for full width details. Obviously, you cannot mix regular and full width within a section.

If section text is missing, all images will be clubbed into one section as a sliding gallery. Section text is a good place to put in project details,outcomes, learning, future scope, etc. Also add accolades.
